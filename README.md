# HUST 计算机图形学实验 - 日地月三体运动模型

这是一个为华中科技大学（HUST）计算机图形学课程设计的实验项目，使用 C++ 和现代 OpenGL (4.4 Core Profile) 从零开始，构建了一个包含太阳、地球和月亮的动态三体运动仿真系统。

## 主要特性

- **精确的运动模型**:
  - 地球沿椭圆轨道绕太阳公转，月亮沿椭圆轨道绕地球公转。
  - 月球的公转轨道相对于地球公转轨道有 **20° 的倾角**，实现了非共面运动。
  - 天体运动速度由各自的公转周期决定，比例关系合理。

- **真实感渲染**:
  - 基于 **Blinn-Phong 光照模型**，计算环境光、漫反射和镜面高光。
  - 为所有天体（太阳、地球、月亮）应用了独立的高分辨率**纹理贴图**。
  - 创新的**双光源系统**：
    1.  一个位于太阳中心的**主光源**，照亮地球和月亮。
    2.  一个位于太阳右后上方的**专属补光灯**，用于照亮太阳自身，增强其立体感。

- **程序化生成**:
  - 所有天体（球体）的顶点、法线、纹理坐标均通过参数方程在程序中动态生成。
  - 天体运行的轨道线也由程序动态生成。

- **交互式体验**:
  - 支持**自由摄像机**，用户可通过键盘在场景中自由漫游。
  - 支持**渲染模式切换**（填充模式 / 线框模式），便于观察模型结构。

## 构建与运行

- **环境**: Visual Studio 2022
- **依赖库**:
  - `Glad` (OpenGL 加载库)
  - `GLFW` (窗口与输入管理)
  - `stb_image` (纹理图像加载)
  - `vmath` (向量与矩阵数学运算)

将项目克隆到本地后，请确保所有依赖库的路径在 Visual Studio 项目属性中配置正确。然后，使用 Visual Studio 2022 打开 `.sln` 文件并编译运行。

## 控制说明

| 按键 | 功能 |
| :--- | :--- |
| `↑` `↓` `←` `→` | 移动摄像机位置 |
| `W` / `S` | 向前 / 向后移动摄像机 |
| `1` | 切换为线框模式 |
| `2` | 切换为填充模式 |
| `ESC` | 退出程序 |
